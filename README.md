# imaotai_autobuy

> 🍾 i茅台自动申购工具 — v0.2.0

## 功能概览

- **API 客户端**：完整封装 i茅台 API，含签名、重试、日志
- **申购核心**：多账号并发申购，支持多商品同时申购，自动选择最优门店
- **申购重试**：申购失败自动重试（最多 3 次，间隔 1 秒），提高成功率
- **定时调度**：精确到秒的申购任务（09:00），随机时间旅行任务
- **结果通知**：通过 Server酱推送申购结果，Token 过期自动告警
- **结果持久化**：每日申购结果自动保存到本地 JSON 文件（按日期归档）
- **健康检查**：启动时验证配置文件完整性，缺失关键字段给出明确提示
- **登录工具**：手机号+验证码获取 Token，写入账号配置

## 快速开始

```bash
# 安装依赖
pip install -r requirements.txt

# 登录获取 Token（交互式）
python tools/login.py

# 启动申购调度
python main.py
```

## 配置文件

`config/config.yaml` — 统一配置（账号、商品、推送设置）

### 多商品配置

每个账号可独立配置商品列表，也可使用全局默认：

```yaml
accounts:
  - mobile: "13800000001"
    token: "..."
    device_id: "..."
    user_id: "..."
    items:           # 每个账号可配置不同商品
      - "10056"      # 飞天茅台
      - "10016"      # 茅台1935

item_code: "10056"   # 全局默认商品ID
city_code: "520100"  # 全局默认城市代码
```

## 项目结构

```
api/          # API 客户端封装
core/         # 申购核心逻辑、账号管理、通知、健康检查
scheduler/    # 定时任务（schedule）
tools/        # 登录工具
utils/        # 签名、时间同步、日志、通知器
data/         # 申购结果持久化（按日期归档 JSON）
tests/        # 105 个单元测试（pytest）
docs/         # 设计文档
```
