# v0.3.0 优化计划

## 背景

v0.2.0 已完成多商品支持、申购重试、结果持久化、Token 过期检测和健康检查。测试覆盖率提升至 105 项。

## Todo-List

- [ ] **Token 自动续期**：集成 API 实现 token 过期时自动刷新，无需手动重新登录
- [ ] **智能门店选择**：根据历史中签率、距离、出货量综合评分选择最优门店
- [ ] **申购统计面板**：基于 data/results/ 中的历史数据，生成申购成功率统计报告
- [ ] **多城市支持**：每个账号可配置不同城市代码，支持跨城市申购
- [ ] **配置热加载**：运行时检测配置文件变更，无需重启即可生效
- [ ] **Docker 部署**：提供 Dockerfile 和 docker-compose.yml，一键部署
