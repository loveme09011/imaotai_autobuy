# v0.1.0 优化计划

## 背景
v0.1.0 完成了基础申购功能框架，但尚未完成真机测试。当前缺少登录凭证获取工具，用户无法便捷配置账号。

## 待开发内容

### 1. 登录工具（高优先级）
- **问题：** 用户无法通过抓包获取 token（i茅台有 SSL Pinning 检测）
- **方案：** 实现 `tools/login.py`，通过官方登录流程（手机号+验证码）直接获取 token
- **Todo-List：**
  - [ ] 实现 `tools/login.py` 命令行登录工具
  - [ ] 支持多账号追加
  - [ ] 自动写入 config/config.yaml
  - [ ] 单元测试
  - [ ] 更新 README.md 使用说明

### 2. 签名算法验证（高优先级）
- **问题：** 当前签名算法基于社区逆向，未经真实请求验证
- **方案：** 登录工具完成后，用真实账号发起申购，验证签名是否被服务器接受
- **Todo-List：**
  - [ ] 真机测试登录接口
  - [ ] 真机测试申购接口
  - [ ] 根据实际返回修正签名算法

### 3. 错误信息优化
- **问题：** 业务错误码未做详细映射
- **方案：** 整理 i茅台常见错误码并映射为用户友好提示
  - [ ] 整理常见错误码
  - [ ] 更新 api/client.py 错误处理
