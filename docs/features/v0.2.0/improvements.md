# v0.2.0 优化计划

## 背景

v0.1.x 已完成全部基础功能，包括 API 客户端、申购核心逻辑、定时调度、结果通知和登录工具。测试覆盖率良好（78 项全部通过）。

## Todo-List

- [x] **多商品支持**：当前只支持单一商品 ID，扩展为支持配置多个商品同时申购
- [x] **重试机制**：申购失败时自动重试（最多 3 次，间隔 1 秒），提高成功率
- [x] **申购结果持久化**：将每日申购结果写入本地 JSON 日志文件（按日期归档）
- [x] **Token 过期自动刷新**：检测 token 失效并提示用户重新登录（发送 Server酱通知）
- [x] **健康检查**：程序启动时验证配置文件完整性，缺失关键字段时给出明确错误提示
- [x] **单元测试补充**：为多商品、重试机制、结果持久化新增测试（78→105）
- [x] **文档更新**
